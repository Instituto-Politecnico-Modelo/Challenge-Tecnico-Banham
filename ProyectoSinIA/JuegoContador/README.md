# JuegoContador

Juego simple creado en React para un challenge técnico. Consiste en clickear un botón la mayor cantidad de veces durante 5 segundos.

## Cómo correr esto en tu computadora

1. Descargá el código o clonalo usando Git
2. Instalá las dependencias con \
pm install\
3. Levantá el proyecto ejecutando \
pm run dev\

Consideraciones:
- Hay que esperar al 'Ya' para que el boton active sus funciones.
- Usé react basico sin hooks personalizados, solo useState y useEffect.
